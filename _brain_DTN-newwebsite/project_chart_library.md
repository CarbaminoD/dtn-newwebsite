Chart library of choice: **ECharts** (via `echarts-for-react`)

Reference site: tpso.go.th uses it — committee approved the look as a benchmark.

**Why:** Committee explicitly called out tpso.go.th charts as the visual target; ECharts is open-source and considered better-looking than amCharts in this context.
**How to apply:** When suggesting data visualization, recommend ECharts / `echarts-for-react`. Avoid recommending amCharts.
