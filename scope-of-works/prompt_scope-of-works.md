# Scope of Works — โครงการพัฒนาเว็บไซต์กรมเจรจาการค้าระหว่างประเทศ ปี 2570

## Context

ฉันคือ Senior Project Manager ของบริษัทฯ ที่ปรึกษาที่ได้รับมอบหมายให้เขียนขอบเขตงานพัฒนาเว็บไซต์กรมเจรจาการค้าระหว่างประเทศ ปี 2570

ข้อมูลที่เกี่ยวข้อง ได้แก่ Related Documents, Website References และ Structure of Scope of Works โดยให้อ้างอิงเนื้อหาหลักจาก `TOR_DTN_เว็บไซต์กรมเจรจาการค้าระหว่างประเทศ_2565.pdf` และปรับปรุงเพิ่มเติมฟังก์ชันใหม่ตามที่หน่วยงานต้องการ

เว็บไซต์ใหม่ที่ต้องพัฒนา ได้แก่
- เปรับปรุงพัฒนาเว็บไซต์กรมเจรจาการค้าระหว่างประเทศ (www.dtn.go.th) ตามภาคผนวก ก
- ปรับปรุงพัฒนาระบบ Intranet กรมเจรจาการค้าระหว่างประเทศ (intranet.dtn.go.th) ตามภาคผนวก ข
- ปรับปรุงพัฒนาระบบบริหารจัดการองค์ความรู้ (KM) (km.dtn.go.th) ตามภาคผนวก ค

> **หมายเหตุ:** เนื้อหาและภาษาในเอกสารขอบเขตงานต้องอยู่ในรูปแบบ **เอกสารราชการ** (Formal/Official Tone)

## Structure of Scope of Works

โครงสร้างอ้างอิงจาก TOR 2565 (13 หัวข้อหลัก + ภาคผนวก) โดยปี 2570 ให้คงโครงเดิมและ **เพิ่ม/ปรับฟังก์ชันใหม่** ตามที่หน่วยงานต้องการ (ทำเครื่องหมาย `*NEW*` หรือ `*ปรับปรุง*` กำกับรายการที่เปลี่ยนจากปี 2565)

| ข้อ | หัวข้อ | หมายเหตุสำหรับปี 2570 |
|---|---|---|
| ๑ | หลักการและเหตุผล | ปรับบริบทให้เป็นปี 2570 อ้างอิงผลแบบสำรวจความพร้อมรัฐบาลดิจิทัล 2569 |
| ๒ | วัตถุประสงค์ | คง 4 ข้อเดิม (PDPA, เชื่อมโยงข้อมูลภายนอก, โครงสร้างพื้นฐาน, KM) + เพิ่มตามฟังก์ชันใหม่ |
| ๓ | คุณสมบัติของผู้เสนอราคา | คงเดิม ทบทวนทีมงาน/ผลงานขั้นต่ำ/วงเงินผลงาน ให้เหมาะกับปี 2570 |
| ๔ | ขอบเขตการดำเนินงาน | หัวใจของเอกสาร — ข้อ ๔.๑๖–๔.๑๘ คือ 3 ระบบหลัก (เว็บไซต์/Intranet/KM) ชี้ไปภาคผนวก ก/ข/ค |
| ๕ | การยื่นเอกสารเสนอราคา | ตารางเปรียบเทียบขอบเขตงาน + การนำเสนอข้อเสนอทางเทคนิค |
| ๖ | ระยะเวลาดำเนินการ | ปี 2565 = 270 วัน (ทบทวนตามขอบเขตใหม่) |
| ๗ | งบประมาณ | ปี 2565 = 3,000,000 บาท (ทบทวนตามขอบเขตใหม่) |
| ๘ | การส่งมอบงานและการจ่ายเงินค่าจ้าง | แบ่งงวด (20/50/30) ทบทวนให้ครอบคลุมฟังก์ชันใหม่ |
| ๙ | เงื่อนไขทั่วไป | ลิขสิทธิ์ Source Code, การรักษาความลับ, SLA แก้ไขปัญหา |
| ๑๐ | การรับประกันผลงาน | รับประกันไม่น้อยกว่า 1 ปี + บำรุงรักษา |
| ๑๑ | หลักเกณฑ์และสิทธิในการพิจารณา | Price Performance (ราคา 30 : คุณภาพ 70) |
| ๑๒ | ข้อสงวนสิทธิของกรมฯ | ลิขสิทธิ์ผลงานเป็นของกรมฯ |
| ๑๓ | ผู้รับผิดชอบโครงการ | ศูนย์สารสนเทศการเจรจาการค้าระหว่างประเทศ |

**ภาคผนวก** (รายละเอียดฟังก์ชันรายระบบ — เทียบ AS-IS จากไฟล์ `DTN Module Concept (Legacy Web).xlsx` และเพิ่ม TO-BE):
- **ภาคผนวก ก** — เว็บไซต์กรมฯ (www.dtn.go.th): 24 หัวข้อใน TOR 2565 (CMS, ผู้ใช้งาน, Template, เมนู, สไลด์, FAQ, แบนเนอร์, Search, ปฏิทิน, Poll, Newsletter, Popup, ฟอร์ม, Gallery, Intropage, Sitemap, Stat, Data Migration, 2 ภาษา) + ฟังก์ชันใหม่ปี 2570
- **ภาคผนวก ข** — ระบบ Intranet (intranet.dtn.go.th): โครงสร้างฟังก์ชันคล้าย ก ปรับสำหรับใช้งานภายใน
- **ภาคผนวก ค** — ระบบ KM (km.dtn.go.th): KM Home, My Portfolio, DTN Knowledge Tank, Knowledge by Job, Photo Gallery, KM Report, KM Channel ฯลฯ

> **ข้อกำหนดทางเทคนิคที่ต้องคงไว้/ปรับ:** Responsive, WCAG 2.0 AAA, มาตรฐานเว็บไซต์ภาครัฐ, PDPA, พ.ร.บ.คอมพิวเตอร์, IPv4/IPv6, SSL Wildcard, SSO (MOC Account), เชื่อมโยง MOC Event — และสำหรับปี 2570 ใช้ tech stack ตาม [Technical Requirements](#) ด้านล่าง (Next.js + .NET Core 10, ECharts)
